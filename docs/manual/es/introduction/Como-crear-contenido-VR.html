[name]

Esta guía proporciona una breve descripción general de los componentes básicos de una aplicación de realidad virtual basada en web creada con three.js.

Workflow

Primero, debe incluir [link:https://github.com/mrdoob/three.js/blob/master/examples/jsm/webxr/VRButton.js VRButton.js] en su proyecto.

import { VRButton } from 'three/addons/webxr/VRButton.js';

*VRButton.createButton()* hace dos cosas importantes: crea un botón que indica compatibilidad con VR. Además, inicia una sesión de VR si el usuario activa el botón. Lo único que tiene que hacer es agregar la siguiente línea de código a su aplicación.

document.body.appendChild( VRButton.createButton( renderer ) );

A continuación, debe indicarle a su instancia de `WebGLRenderer` que habilite el renderizado XR.

renderer.xr.enabled = true;

Finalmente, debe ajustar su ciclo de animación ya que no podemos usar nuestro conocida función *window.requestAnimationFrame()* . Para proyectos de realidad virtual (VR) usamos [page:WebGLRenderer.setAnimationLoop setAnimationLoop]. El código mínimo se ve así:

renderer.setAnimationLoop( function () { renderer.render( scene, camera ); } );
