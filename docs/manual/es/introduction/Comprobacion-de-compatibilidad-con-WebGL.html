Comprobación de compatibilidad con WebGL

Aunque esto se está convirtiendo en un problema cada vez menor, es posible que algunos dispositivos o navegadores aún no sean compatibles con WebGL. El siguiente método le permite verificar si es compatible y mostrar un mensaje al usuario si no lo es.

Agregue [link:https://github.com/mrdoob/three.js/blob/master/examples/jsm/capabilities/WebGL.js] a su javascript y ejecute lo siguiente antes de intentar renderizar algo.

if ( WebGL.isWebGLAvailable() ) { // Inicie la función u otras inicializaciones aquí animate(); } else { const warning = WebGL.getWebGLErrorMessage(); document.getElementById( 'container' ).appendChild( warning ); }
