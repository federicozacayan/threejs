[name]

Clase que representa objetos basados en [link:https://en.wikipedia.org/wiki/Polygon_mesh polygon mesh] triangulares. También sirve como base para otras clases como SkinnedMesh. Also serves as a base for other classes such as [page:SkinnedMesh].

Ejemplo de código

const geometry = new THREE.BoxGeometry( 1, 1, 1 ); const material = new THREE.MeshBasicMaterial( { color: 0xffff00 } ); const mesh = new THREE.Mesh( geometry, material ); scene.add( mesh );

Constructor

[name]( [param:BufferGeometry geometry], [param:Material material] )

[page:BufferGeometry geometry] — (opcional) una instancia de [page:BufferGeometry]. El valor predeterminado es una nueva [page:BufferGeometry].
[page:Material material] — (optional) (opcional) una sola o una matriz de [page:Material]. El valor predeterminado es un nuevo [page:MeshBasicMaterial]

Propiedades

Consulte la clase base [page:Object3D] para conocer las propiedades comunes.

[property:BufferGeometry geometry]

Una instancia de [page:BufferGeometry] (o clases derivadas), que define la estructura del objeto.

[property:Boolean isMesh]

solo lectura: Indicador para verificar si un objeto dado es del tipo [name].

[property:Material material]

Una instancia de material derivado de la clase base [page:Material] o una matriz de materiales, que define la apariencia del objeto. El valor predeterminado es [page:MeshBasicMaterial].

[property:Array morphTargetInfluences]

Una matriz de ponderaciones normalmente de 0 a 1 que especifica la cantidad de transformación que se aplica. Indefinido de forma predeterminada, pero restablecido a una matriz en blanco por [page:Mesh.updateMorphTargets updateMorphTargets].

[property:Object morphTargetDictionary]

Un diccionario de morphTargets basado en la propiedad morphTarget.name. Indefinido de forma predeterminada, pero reconstruido [page:Mesh.updateMorphTargets updateMorphTargets].

Métodos

Consulte la clase base [page:Object3D] para conocer los métodos comunes.

[method:Mesh clone]()

Devuelve un clon de este objeto [nombre] y sus descendientes.

[method:undefined raycast]( [param:Raycaster raycaster], [param:Array intersects] )

Obtenga las intersecciones entre un rayo proyectado y esta malla. [page:Raycaster.intersectObject] llamará a este método, pero los resultados no están ordenados.

[method:undefined updateMorphTargets]()

ctualiza los morphTargets para que no tengan influencia en el objeto. Restablece el [page:Mesh.morphTargetInfluences morphTargetInfluences] y la propiedad [page:Mesh.morphTargetDictionary morphTargetDictionary].
