[name]

Escenas le permite configurar qué y dónde debe renderizar three.js. Aquí es donde colocas objetos, luces y cámaras.

Constructor

[name]()

Crea un nuevo objeto de escena.

Properties

[property:Object background]

Si no es nulo, establece el fondo utilizado al renderizar la escena y siempre se renderiza primero. Se puede establecer en [page:Color] que establece el color claro, una [page:Texture] que cubre el lienzo, un mapa de cubos como [page:CubeTexture] o un equirrectangular como [page:Texture] . El valor predeterminado es nulo.

[property:Texture environment]

Si no es nula, esta textura se establece como mapa de entorno para todos los materiales físicos de la escena. Sin embargo, no es posible sobrescribir una textura existente asignada a [page:MeshStandardMaterial.envMap]. El valor predeterminado es nulo.

[property:Fog fog]

A [page:Fog fog] instance defining the type of fog that affects everything rendered in the scene. Default is null.

[property:Boolean isScene]

Solo lectura: Indicador para verificar si un objeto dado es del tipo [name].

[property:Material overrideMaterial]

Si no es nulo, forzará que todo en la escena se renderice con ese material. El valor predeterminado es nulo.

Métodos

[method:Object toJSON]( [param:Object meta] )

meta - objeto que contiene metadatos como texturas o imágenes para la escena.
Convierte la escena a three.js [link:https://github.com/mrdoob/three.js/wiki/JSON-Object-Scene-format-4 JSON Object/Scene format].
