El primer paso consiste en crear un archivo `index.html` que reúna todos los archivos necesarios para comenzar.

NOTA

Este es el primero de los 9 pasos del tutorial práctico, moderno, escalabe y profesional de `three.js` que se ofrece con un enfoque completamente diferente al tutorial de la documentación oficial. Al finalizar cada uno de los pasos, encontrarás una referencia al próximo paso.

Código

Se asume que se cuenta con conocimientos básicos de `HTML`, `CSS`, y `JavaScript`

<!DOCTYPE html> <html> <head> <title>Tutorial de three.js</title> <style> body { margin: 0; } </style> <script src="js/three.js"></script> <script src="src/index.js" type="module"></script> </head> <body> </body> </html>

CSS

Se puede colocar el `CSS` en otro archivo. Por ejemplo, en una subcarpeta con el nombre `css`:

<!DOCTYPE html> <html> <head> <title>Tutorial de three.js</title> <link rel="stylesheet" href="css/style.css"> <script src="js/three.js" defer></script> <script src="src/index.js" type="module"></script> </head> <body> </body> </html>

En cuyo caso el contenido del archivo `css/style.css` sería el siguiente

body { margin: 0; }

Subcarpetas con archivos Javascript

Será necesario crear dos subcarpetas. La subcarpeta `/js` y la subcarpeta `/src`.

La primera de las subcarpetas tendrá la libreria `three.js` que se podrá obtener [link:https://threejs.pro/build/three.js aqui]. Y la misma se ejecutará luego de cargar todo el HTML debido al atributo `defer` de la etiqueta `script`.

La segunda de las subcarpetas contendrá el codigo de la aplicación, incluyendo el archivo `entry point` con el nombre `index.js` donde comenzará el hilo de la ejecución de la misma. Este archivo también se ejecutará luego de la carga de todo el HTML y después de la ejecución del código de la librería.

A diferencia del código de la librería, el archivo `index.js` podrá utilizar la sintaxis de los módulos `EcmaScript` debido a que se en la etiqueta script se incluyó el atributo `type="module"`.
