[name]

Es una instancia de la clase `THREE.Mesh()` , que recibe dos parámetros: `geometry` y `material`.

La ubicación de este módulo es `src/basic/shapes/Cube.js`.

`mesh` significa maya en español.

Código

Nuevamente se vuelve a encapsular la complejidad de la construcción de la instancia dentro del módulo y se exporta simplemente la instancia, que queda lista para ser usada.

const geometry = new THREE.BoxGeometry(1, 1, 1); const material = new THREE.MeshStandardMaterial({ color: 0x00ff00 }); const cube = new THREE.Mesh(geometry, material); export default cube

Modulo Cube

En pocas palabras el `cubo` se construye con una `geometría` y un `material`.

Una `geometría` es un conjunto de puntos distribuidos en el espacio con el que se formarán triángulos.

Por otro lado, un `material` es una instancia de una clase que contiene información de cómo se pintarán los triángulos de la `maya`.
