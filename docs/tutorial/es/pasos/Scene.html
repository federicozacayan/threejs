Se trata de la instancia de un la clase `THREE.Scene()` que se coloca en el módulo `src/basic/Scene.js`.

Una vez que se tienen los módulos básicos en la carpeta `basic`, simplemente se los puede copiar y pegar de proyecto en proyecto copiando la carpeta `basic`.

Módulo Scene

Es bastante simple y no hay mucho para decir de este módulo.

const scene = new THREE.Scene(); export default scene

Import y Export

Como se puede ver simplemente se utilizan las palabras claves `export default` para exportar la instancia.

Ya se vió en el paso anterior como se importa esta instancia con las palabras claves `import` y `from`.
