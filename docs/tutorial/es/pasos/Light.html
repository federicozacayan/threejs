Este es un módulo un poco complejo. Se trata de una instancia de la clase `THREE.AmbientLight`. Que a su vez, dentro de esta misma instancia se coloca otra instancia distinta, pero de la clase `THREE.DirectionalLight`.

Esto se puede hacer debido a que tanto la `luz ambiental` como la `luz direccional` extienden de la clase `Object3D`, la cual permite a travez del método `.add()` agregar otros Object3D como hijos del mismo Object3D.

Es una manera organizada pero poco elegante de tener un único módulo con todo lo relacionado a las luces.

La ubicación de este módulo es `src/basic/Light.js`.

Código

Nuevamente se vueve a exportar la instancia con las palabras claves `export default`.

const ambientLight = new THREE.AmbientLight(0x404040); // soft white light const directionalLight = new THREE.DirectionalLight(0xffffff, 0.5); directionalLight.position.set(-10, 10, 10) ambientLight.add(directionalLight) export default ambientLight

Exportar un módulo por defecto

Notar que el nombre de la instancia es `ambientLight` pero en el paso `EntryPoint` la instancia de la luz se llama `light`.

Esto es posible debido a que cuando se exporta un objeto usando la palabra clave `default` se permite utilizar un alias sin ninguna sintaxis especial.
