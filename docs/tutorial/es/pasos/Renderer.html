El módulo `Renderer` es el módulo central de todo el sistema. Es una instancia de la clase `THREE.WebGLRenderer()`.

La ubicación de este módulo es `src/basic/Renderer.js`.

Código

En este módulo, además de instanciar la clase `THREE.WebGLRenderer`, se configura las dimenciones del canvas y se coloca dicho canvas dentro del `body` del archivo `html`.

const renderer = new THREE.WebGLRenderer(); renderer.setSize(window.innerWidth, window.innerHeight); document.body.appendChild(renderer.domElement); export default renderer

Módulo Renderer

Como se ha visto en los pasos anteriores, se exporta la instancia con las palabras claves `export default`.

Adicionalmente, se generan algunas configuraciones, pero quedan encapsuladas en el mismo módulo y son totalmente transparentes de su implementación.

Es decir, en la implementacion no se visualiza la complejidad del módulo y queda oculta dentro del mismo módulo.
