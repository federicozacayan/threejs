Código

En la primera parte de este archivo se importan diferentes módulos `EcmaScript`.

En la segunda parte del archivo está el código de la aplicación.

import scene from "./basic/Scene.js"; import camera from "./basic/Camera.js"; import renderer from "./basic/Renderer.js"; import cube from "./basic/shapes/Cube.js"; import loopMachine from "./basic/LoopMachine.js"; import resize from "./basic/Resize.js"; import light from "./basic/Light.js"; scene.add(cube); scene.add(light) camera.position.set(2,2,2) camera.lookAt(cube.position) loopMachine.addCallback(() => { cube.rotation.y += 0.01; renderer.render(scene, camera); }) resize.start(renderer) loopMachine.start()

Módulos

Los objetos utilizados aquí, tales como `scene`, `cube`, `light`, `camera`, `loopMachine`, `renderer` y `resize` provienen de otros archivos.
