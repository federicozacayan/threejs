Este es un módulo encargado de ajustar el tamaño del canvas cuando la ventana cambia de tamaño.

La ubicación de este módulo es `src/basic/Resize.js`.

Código

Este módulo depende de la `camara` y del `renderizador`.

A la instancia de la `camara` se accede utilizando `imports`.

A la instancia del `renderizador` se accede a travez de la inyeción del objeto en el método `.start()`

La `camara` debería insertarse como se inserta el `renderizador`, pero no es un gran problema hacerlo de esta manera.

import camera from "./Camera.js" class Resize { constructor() { this.renderer = null } start(renderer) { this.renderer = renderer window.addEventListener('resize', this.resize) } stop() { window.removeEventListener('resize', this.resize) } resize = () => { camera.aspect = window.innerWidth / window.innerHeight camera.updateProjectionMatrix() this.renderer.setSize(window.innerWidth, window.innerHeight); } } const resize = new Resize() export default resize
